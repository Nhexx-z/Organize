# Folder Organizer & Stats

A simple application that helps organize folders and generate statistics about the files they contain.

## Features

- Organize files inside folders
- Analyze folder contents
- Generate statistics about:
  - File types
  - File sizes
  - File distribution

## Requirements

- Python 3.x
